[CSS] padding & margin

Posted on 2020-07-20 In Frontend , CSS
Symbols count in article: 502 Reading time ≈ 1 mins.

Padding

設定定義元素邊框與元素內容間的距離。

可以發現 box1 元素的字距離邊框有 30px 的邊距

其他寫法

  • padding 四個邊都是 30px。
  • 上填充為25px
  • 右填充為50px
  • 下填充為75px
  • 左填充為100px
  • 上填充為25px
  • 左右填充為50px
  • 下填充為75px
  • 上下填充為25px
  • 左右填充為50px

另外有針對四個邊的獨立寫法：

  • padding-top：上方的內距
  • padding-right：右方的內距
  • padding-bottom：下方的內距
  • padding-left：左方的內距
    四個屬性可以單獨使用也可以混搭使用。

Margin

設定元素邊框與外面的距離。

其他寫法

margin 的其他寫法與 padding 完全相同。

  • margin-top：設置元素的上外邊距。
  • margin-right：設置元素的右外邊距。
  • margin-left：設置元素的左外邊距。
  • margin-bottom：設置元素的下外邊距。
    針對單一的 margin 屬性也可以設 一個到四個 的值。
